File Descriptions:
PA1.cpp -- The main file, uses all the other files to launch the Math Wizard application.
MathReport.cpp - Provides the implementation of member functions for the class MathReport
MathReport.h - Declares the class named MathReport
MathOperations.cpp - File used to provide the implementation of the class named MathOperations.
MathOperations.h - Declares the class MathOperations
How to compile source code: First, make sure you are in the working directory (in my case, its /CS216/Project1). Then type g++ PA1.cpp MathOpertions.cpp MathReport.cpp -o Project1
Then if you want to launch the application, type ./Project1
